# AI Internship - Day 3 Git Practice

## Objective
Learn Git and GitHub workflow through hands-on practice.

## Git Commands Practiced

- git init
- git add
- git commit
- git push
- git pull
- git clone
- git branch
- git merge

## Professional Commit Messages

- feat
- docs
- refactor
- chore

## Workflow Followed

1. Initialized Git repository.
2. Added project files.
3. Created meaningful commits.
4. Created feature/python branch.
5. Pushed branch to GitHub.
6. Merged feature branch into main.
7. Updated GitHub repository successfully.

## Outcome

Successfully learned and practiced Git version control and GitHub collaboration workflow.
